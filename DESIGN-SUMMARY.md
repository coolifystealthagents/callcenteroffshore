# Design summary

- Design concept: Industrial operations
- Layout personality: grid-heavy command board
- Typography: Roboto Condensed headings with IBM Plex Sans body text
- Color palette: primary #1f2937, secondary #f97316, accent #0ea5e9, background #f3f4f6, text #111827, muted #6b7280
- Shape language: Sharp corners
- Visual motif: technical grid lines and status tags
- Unique layout choices: first sections use SLA dashboard, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
